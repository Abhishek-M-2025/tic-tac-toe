## ✨ Features

- 🎯 Two-player gameplay (X vs O)
- 🏆 Winner detection
- 🤝 Draw detection
- 🔄 Reset Game button
- 🎨 Responsive and modern dark UI
- 🖱️ Hover effects and player color indicators

## 🛠️ Technologies Used

- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript (Vanilla JS)

## 📚 Concepts Used

- DOM Manipulation
- Event Listeners
- Arrays
- Functions
- Conditional Statements (`if...else`)
- Loops (`for`)
- CSS Grid
- Flexbox
- CSS Hover Effects
